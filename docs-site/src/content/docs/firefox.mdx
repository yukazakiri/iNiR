# Firefox

iNiR integrates with Firefox (and other Gecko-based browsers) to dynamically sync your browser theme with the system Material You colors. It achieves this by generating configuration files designed to be consumed by the **Pywalfox** extension.

## Documentation

Instead of directly interacting with Firefox CSS internals, iNiR generates a compliant JSON file whenever a wallpaper or theme is applied.

1. `scripts/colors/apply-gtk-theme.sh` extracts primary colors from `colors.json`.
2. It calculates the correct background (`$BG`), foreground (`$FG`), and primary (`$PRIMARY`) colors.
3. It creates a `pywalfox-colors.json` file in `~/.local/state/quickshell/user/generated/`.

## Setup

To enable Firefox theming, you need to set up the Pywalfox extension and companion script:

1. Install the [Pywalfox extension](https://addons.mozilla.org/en-US/firefox/addon/pywalfox/) in your browser.
2. Ensure you have the Pywalfox companion script installed on your system (e.g., via `pip` or your package manager).
3. The companion script needs to know where your `pywalfox-colors.json` file lives so it can sync the changes back into Firefox.

Once configured, Pywalfox will automatically detect changes to the JSON file generated by iNiR and immediately apply your new background, foreground, and tab colors without restarting Firefox.
