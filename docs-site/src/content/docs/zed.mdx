# Zed Editor

Zed is a high-performance, multiplayer code editor from the creators of Atom and Tree-sitter. iNiR provides automatic theming capabilities for Zed.

## Documentation

iNiR ships with a built-in Python script (`scripts/colors/zed/theme_generator.py`) that automatically generates a fully customized Zed theme. 

### How it works

1. The script reads your generated Material You colors from `~/.local/state/quickshell/user/generated/colors.json`.
2. It also extracts terminal colors generated by `matugen` via the `material_colors.scss` file.
3. It performs intelligent color manipulation (adjusting lightness, saturation, and luminance) to ensure accessibility, syntax highlighting contrast, and aesthetic consistency. 
4. The generator dynamically produces two theme variants:
    - **iNiR Dark**: Uses deeply saturated surface and primary colors tailored for low-light environments.
    - **iNiR Light**: Intelligently lightens and adjusts colors while preserving legibility and terminal contrast.
5. It exports a compliant JSON file ready to be loaded directly by Zed.

### Installation

1. First, ensure the Zed theme output path is active and accessible to Zed's theme extensions directory (`~/.local/share/zed/themes/` or similar, depending on your OS).
2. The generated theme will be named **"iNiR Material"** containing two sub-themes:
   - `iNiR Dark`
   - `iNiR Light`
3. Open your Zed `settings.json` (`~/.config/zed/settings.json`) and apply the desired theme:

```json
{
  "theme": "iNiR Dark"
}
```

*Note: Whenever your system wallpaper/theme updates via iNiR, the generation script runs automatically and updates the JSON file. You may need to reload or refocus your Zed window for changes to appear.*
